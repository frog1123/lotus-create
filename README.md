# lotus-create

<div align="center">

a tool for generating basic javascript or typescript project

![NPM Downloads](https://img.shields.io/npm/dw/lotus-create)

</div>

## getting started

run the command

```
npx lotus-create
```

or install it globally

```
npm i -g lotus-create
```
